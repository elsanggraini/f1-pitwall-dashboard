# 🏎️ F1 Smart Pit Wall Telemetry Dashboard

Futuristic IoT telemetry dashboard inspired by Formula 1 pit wall systems using ESP32 + DHT11.

## 🚀 Features

- Real-time temperature monitoring
- Real-time humidity monitoring
- Dynamic F1-inspired UI
- Smart tire strategy recommendation
- Driver fatigue risk analysis
- Smooth live telemetry graphs
- Responsive web dashboard hosted directly from ESP32

---

## 🧠 Technologies Used

- ESP32
- DHT11 Sensor
- HTML/CSS/JavaScript
- Chart.js
- Arduino IDE
